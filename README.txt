Web visitor counter help to show page or post view count to site visitor.

== Description ==

Web visitor counter plugin rely on Jetpack to show page or post view count. Please make sure you install Jetpack to get the correct visitor count data.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Install 'Web Visitor Counter' plugin 
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add "page_view" shortcode to the post or page you wish to show view counts.


== Frequently Asked Questions ==

= If without Jetpack will this plugin function correctly? =

No. Because the view count data is taken from Jetpack.

= How to make this plugin work? =

Add shortcode of "page_view" to the post or page you want to show view count.

= How to change css? =

Use class "ast-page-view" to change the css to the design that match your website.
